## Suggested README Positioning

This is not another coding agent.
It is a memory and verification layer for the agents you already use.

## Obsidian Positioning

The user installs Obsidian, opens `starter_vault/` as a vault, and gives the vault path to an AI agent.
The vault can be localized later by asking the agent to rewrite vault-facing project documents in the user's preferred language while preserving stable slugs and technical fields.
