This hub collects the public, reusable Agent Guardian charters.
Project folders should link to project memory, not duplicate these charters unless the user explicitly wants a project-local fork.

## Project Rule

New projects should reference this hub and keep their own notes project-prefixed.
Do not copy every charter into every project folder by default.

Copy a charter into a project only when:

- the project needs a specialized fork
- the copy is clearly named with the project prefix
- the project registry records why the fork exists
- the global charter remains unchanged
