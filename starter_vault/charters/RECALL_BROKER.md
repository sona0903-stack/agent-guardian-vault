# RECALL_BROKER

Recall is a controlled search process, not a full-vault dump.

## Before Solving A Repeated Problem

Search the relevant project notes first:

- master
- handoff
- decisions
- errors
- current part

Then search shared skills and charters if needed.

## Evidence Split

When reporting uncertain information, separate:

- confirmed
- inferred
- unknown

Do not blur these categories.

## Cross-Project Recall

Cross-project recall is allowed only when it helps the current task.
Record the source project and note title when a pattern is reused.

Recommended entry:

```text
Reused pattern from: `projects/<source-project>/<source-project>_master.md`
Reason: <why it applies>
Risk: <what may not transfer>
```
