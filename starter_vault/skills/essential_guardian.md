# essential_guardian

## When To Use

Use for every Agent Guardian task.

## Rules

- Preserve-first.
- Read existing files before editing.
- Record the user command before work.
- Keep one active task.
- Apply the coding micro-harness before edits.
- Do not hide failures.
- Verify before completion.
- Record decisions.
- Record errors and failed attempts.
- Promote reusable patterns after repeated use.
- Do not use destructive reset without explicit user approval.

## Verification

- master and part note agree
- files touched are listed or obvious
- proof exists
- unresolved items are visible
