Agent Guardian Vault Starter
GitHub Starter Vault Agent Adapters

Local-first memory for long AI coding projects.

Agent Guardian Vault Starter is an Obsidian-style memory and handoff layer for AI coding agents. It helps Codex, Claude Code, ChatGPT, OpenHands, Cursor, and local LLM agents preserve decisions, errors, proof, and next actions across long sessions.

What It Preserves

Project State

Keep one active task, current status, touched files, proof, and next action visible.

Decision And Error Ledgers

Record why choices were made, what failed, and what should be recalled later.

Handoff Between Chats

Move long-running work across sessions without pasting the entire conversation every time.

How To Use It

Open the starter_vault/ folder as an Obsidian vault, then give that path to your AI agent. The repository includes start prompts for Codex, Claude Code, ChatGPT, OpenHands, and local LLM agents. It is a memory harness, not a runtime installer.
